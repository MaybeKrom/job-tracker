# Job Tracker

I applied to 80+ COOP internships and forgot who half of them were. So I built this.

## What it does
- Paste any job post or email — AI extracts the details automatically
- Chrome extension saves jobs from any website in one click
- Tracks every application in one dashboard
- Status updates (Applied, Got Call, Interview, Offer, Rejected)
- Notes per application that auto-save as you type
- Direct link back to the original job posting

## Stack
- PHP + MySQL
- Claude AI API (extraction)
- Vanilla HTML/CSS/JS
- Chrome Extension (Manifest V3)

## Setup
1. Clone the repo
2. Run on XAMPP
3. Import the database schema in phpMyAdmin
4. Create config.php and add your Claude API key
5. Load the extension folder in chrome://extensions
